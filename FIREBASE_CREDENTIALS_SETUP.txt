How to get firebase_credentials.json and where to put it
========================================================

You must download this file from YOUR Firebase project. It cannot be generated by the app.

Steps:
------
1. Open https://console.firebase.google.com/
2. Select your project: visitor-management-8f5b4
3. Click the gear icon (Settings) next to "Project Overview" → Project settings
4. Open the "Service accounts" tab
5. Click "Generate new private key" → Confirm
6. A JSON file will download (e.g. visitor-management-8f5b4-firebase-adminsdk-xxxxx.json)

Where to put it:
----------------
Rename the downloaded file to:  firebase_credentials.json

Then COPY it into these three folders (same file in each):
  - Register_App\firebase_credentials.json
  - Admin\firebase_credentials.json
  - Webcam\firebase_credentials.json

Create the Realtime Database (fixes "Database unavailable" / 404):
-------------------------------------------------------------------
If the app says "Database unavailable" or Firebase returns 404, the Realtime Database may not exist yet.

Direct link (open in browser):
  https://console.firebase.google.com/u/0/project/visitor-management-8f5b4/database/visitor-management-8f5b4-default-rtdb/data/~2F

1. Open the link above (or: Firebase Console → your project → Build → Realtime Database).
2. If you see "Create Database", click it.
3. Choose a location (e.g. us-central1 for US). Click "Next", then "Start in test mode" for development. Done.
4. On the Realtime Database page, find the database URL at the top (e.g. https://visitor-management-8f5b4-default-rtdb.firebaseio.com or https://...default-rtdb.REGION.firebasedatabase.app). COPY IT EXACTLY.
5. Put that URL in Register_App/.env, Admin/.env, and Webcam/.env as:
   FIREBASE_DATABASE_URL=<paste the URL you copied>
   (No trailing slash, or one trailing slash — both are fine.)
6. Restart Register_App, Admin, and Webcam.

Add one employee and one room (so registration dropdowns are not empty):
-------------------------------------------------------------------------
After the database exists and FIREBASE_DATABASE_URL is set, from the project root run:
  python seed_firebase_data.py
This adds "Admin User" as an employee and "Conference Room A" as a meeting room. Then refresh the registration page.

Important:
----------
- Do NOT commit this file to git (it is already in .gitignore).
- Keep it secret; it gives full access to your Firebase project.
